Warehouse Scheduler

The Warehouse Scheduler is a comprehensive solution designed to manage and optimize warehouse operations. It allows managers to create, edit, and manage tasks efficiently, edit workers availability. System can automatically assign workers for tasks and generate schedules. Workers can check their schedules, ensuring smooth operations.

About Project

This application was developed as a graduation project by Rohoza Sofiia, student of FIOT, IS-03.

Features

  • User management for administrators
  • Tasks and workers management for managers
  • Worker schedule view and updates
  • Real-time task assignment
  • Optimized for performance and reliability
